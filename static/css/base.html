Hakkımızda

Go-st, kullanıcıların ilgi alanlarına ve bütçelerine uygun tatil seçeneklerini kolayca bulabilmeleri amacıyla geliştirilmiş bir tatil planlama platformudur.

Sistem; ülke, tatil türü ve bütçe bilgilerini değerlendirerek kullanıcıya uygun otelleri listelemektedir.

Proje Flask, HTML, CSS ve Google Hotels API kullanılarak geliştirilmiştir.
